Brainy Spelling

  • Description
  • Links

This native Android application was created for the two-semester Georgia Tech Junior Design project class. Using Agile-based Object-Oriented software design priciples, our team worked closely with our client, Dr. Walter Evans, to create a children's spelling app based around learning phonetics.

I was in charge of designing and implementing the backend of the project and used a JSON system to store and organize all the words and associated files for each game mode.

Github Repository:

Shelter App

  • Description
  • Links

This native Android application was created for our Agile Object-Oriented software engineering class at Georgia Tech. We worked with a client-given CSV to display a list of the closest shelters on the Google Maps API. The project was organized by milestones and used a burndown chart was used for each sprint.

I did most of the backend work on this app. I used a MySQL database to allow the user to rent out up to six beds at any shelter with available space. JSONs were used to interface between the native Android application and the MySQL server.

Github Repository:
