Corner Face-Off!

  • Description
  • Links

This game was programmed entirely for the Game Boy Advance hardware in C.

It is about one of my professors, Fisayo Omojokun, becoming the best boxer in the world. He also fights a glitch... Or something... Called Cornerface.

I did this in the span of 3 weeks as a final project for my Media Device Architecture class. I played to the fact that it would be presented live and had a performance aspect to it. There is a part where the game breaks the fourth wall and speaks to me as part of a live act.

The reactions that the class had to this game were absolutely surreal for me. Some students were one the floor from laughing so hard. Everyone who left the class that day personally went up to me and thanked me for making this game. At the time, I didn't really know how to feel. Looking back, though. I couldn't be any happier. This game gave me a lot of lasting friendships.

I worked well into the night like a machine, drawing and programming non-stop. If there was ever a game I was most proud of, it would be this small little demo. It really proved to me that I can do anything I put my mind to, and it showed that I really understood how to program at a fundamental level.

I used a lot of advanced hardware tricks in the GBA. I used Alpha Blending to create transparency for the main character (pictured), Mosaic and Black/White Blending to create transition, and—for the grand finale—I used affine backgrounds to create a zoom effect at the very end of the game. The affine backgrounds used something called Mode 1, which has never been used or taught in the class before. It was a first.

Because of this game, I ended up becoming a Teaching Assistant for this class at my professor's request. My life would be very different without this game; I owe it so much.

Since this game is exclusively for the GBA hardware, you will need an emulator to play this. Since this game uses advanced hardware features, it will ONLY work on Visual Boy Advance - M 2.0.1. Not even later versions work.

Mental Block!

  • Description
  • Links

I created this game entirely from scratch in Unity in the span of 13 days. It's one of the few games that I've made that can stand on its own as a whole experience.

It was created for a homework assignment for my Computational Media Game Design class. The assignment was to create a platformer that included meaningful aesthetics and art. Essentially, all the art in the game has a reason for why it is the way it is.

This was totally up my alley. I decided to make a game that combined platforming with Pokémon Ice Puzzles. I made the main character a psychic and had all the colors relating to psychic energy be purple. I was also really inspired by the game Cave Story when making this.

I made this game mostly for my grandpa. He passed away the week before I started making this game. He was a guy who loved saying "What's a matter, mang?" and was obsessed with red Ferraris. I loosely designed the character SpÜka after him.

This game turned out to have a really cool aesthetic and lovely story. My professor questioned if I actually made this in the project timespan or if I was working on it before the project was assigned. When I told him no, he said that it was what he expected for final project quality and told me to add more levels to submit it to a competition or game jam.

In the words of some of my friends from the class where I presented Corner Face-Off, I really "did it again."
